# Specification

## Summary
**Goal:** Build the Phase 1 shell of the AO Farms Dashboard UI with a themed navigation bar, hero banner, summary stat cards, a 7-day milk production chart, and a recent milk entries panel.

**Planned changes:**
- Add a top navigation bar with the AO Farms logo (green cow icon + "A.O. Farms" text) on the left and links: Dashboard, Cattle, Milk Production, Inventory, Customers, Deliveries — with the active link styled as a filled dark-green pill
- Add a full-width hero banner on the Dashboard with a farm landscape background image and overlaid "Welcome to AO Farms" title + subtitle text
- Display 6 summary stat cards in a row on the Dashboard (Total Cattle, Today's Milk, Inventory Items, Low Stock, Today's Deliveries, Missed Deliveries) each with label, value, subtext, and icon
- Add a "7-Day Milk Production" area/line chart in the lower-left panel of the Dashboard
- Add a "Recent Milk Entries" panel in the lower-right of the Dashboard listing recent milk logs with cattle tag, date, and volume
- Apply a farm-themed visual style: beige/cream page background, dark green nav and accents, warm white cards, earthy typography

**User-visible outcome:** Users see a polished AO Farms dashboard with branded navigation, a farm hero banner, at-a-glance stat cards, a milk production trend chart, and a recent milk entries list.
